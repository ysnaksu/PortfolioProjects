### 1. COVID-19 Data Analysis
**Objective:** Analyze COVID-19 cases, deaths, and vaccination trends across countries and continents.  
**Skills demonstrated:**

- CTEs (Common Table Expressions) for advanced calculations

**Example insights:**  
- Total cases vs population percentages  
- Countries with highest infection and death rates  
- Vaccination progress over time  
- Breakdown by continent

### 2. Data Cleaning Project
**Objective:** Clean and standardize a real estate dataset to make it analysis-ready.  
**Skills demonstrated:**  
- Standardizing date formats  
- Populating missing data using joins  
- Splitting address and owner columns into separate fields (Address, City, State)  
- Transforming Y/N values into Yes/No  
- Removing duplicate records  
- Dropping unused or unnecessary columns  

**File:**  
- `Data Cleaning Portfolio Project.sql`

**Outcome:**  
- A clean, consistent, and analysis-ready dataset suitable for reporting or further analytics.
